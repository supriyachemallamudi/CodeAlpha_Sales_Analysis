CodeAlpha Sales Prediction Project

Overview

This project is developed as part of the CodeAlpha Data Science Internship.

The main objective of this project is to predict sales based on advertising data using Machine Learning with Python.

Technologies Used

Python
Scikit-learn
Matplotlib
Project Description

In this project, a Linear Regression model is used to analyze advertising data and predict sales.

The model is trained using:

TV Advertising
Radio Advertising
Newspaper Advertising
After training, the model predicts future sales values and displays the sales trend graph.
